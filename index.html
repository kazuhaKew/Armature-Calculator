Информация об арматуре

Типы арматуры

  • A-I (A240) - гладкая арматура для несложных конструкций
  • A-III (A400) - рифленая арматура для железобетонных конструкций
  • A500C - современная арматура повышенной прочности
  • A-IV (A600) - высокопрочная арматура для ответственных конструкций

Диаметры арматуры

Стандартные диаметры: 6мм, 8мм, 10мм, 12мм, 14мм, 16мм, 18мм, 20мм, 22мм, 25мм, 28мм, 32мм, 36мм, 40мм

Наиболее распространенные в частном строительстве: 8мм, 10мм, 12мм, 14мм

Полезные советы

  • Для фундаментов частных домов обычно используют арматуру диаметром 10-14мм
  • Для плит перекрытия оптимален диаметр 8-12мм
  • При расчетах учитывайте нахлесты при соединении - обычно 40-50 диаметров арматуры
